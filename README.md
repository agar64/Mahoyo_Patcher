Requires [mahoyo_tools](https://github.com/loicfrance/mahoyo_tools)

Patcher for Mahou Tsukai no Yoru that changes measurements from US Customary Units to Metric. 
In addition it also, optionally, can change the romanization of some names to Hepburn, revert the name order to the native Japanese one, and reintroduce honorifics.
